# Node.js Task3 Project

## Deployment Guide (Render)
1. Create GitHub repository and push this project.
2. Go to https://render.com and create a free account.
3. Click 'New Web Service' and connect your GitHub repo.
4. Build Command:
   npm install
5. Start Command:
   npm start
6. Add environment variables in Render dashboard if using database.

This project is ready to deploy.
